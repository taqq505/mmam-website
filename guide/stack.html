stack

FastAPI + PostgreSQL + Vue 3

docker compose が FastAPI サービス、PostgreSQL 16、MQTT ブローカー、Vue 3 UI をまとめて起動します。UIは vendor/ に含まれ、オフライン環境でも動作します。

FastAPIレイヤー

  • PATCH /flows/{id}・checkerエンドポイント
  • NMOS discover/apply ツール
  • Automationスケジューラと監査ログ

PostgreSQL

  • flows / planner_buckets / address_map
  • alias_1-8, user_field_1-8 でテキストハブ
  • checker結果やAutomation履歴を保存

Vue 3 UI

  • Planner/Explorer、NMOSウィザード、Checker
  • テキストハブ付きフロー編集
  • ショートカット中心の操作感
