setup

初めてでもできるセットアップ

Git や Docker を初めて触る方に向けて、コマンドと画面操作を丁寧に解説します。各ステップごとに確認ポイントも記載しています。

トップページへ READMEを見る

ステップバイステップ

  1. リポジトリを取得 — git clone https://github.com/taqq505/mmam-docker.git
  2. .envを作成 — cp .env.example .env 後、POSTGRES_PASSWORD や SECRET_KEY を任意値に変更。
  3. Dockerを起動 — docker compose up --build。初回は依存イメージの取得で数分かかります。
  4. ブラウザで確認 — http://localhost:4173 または https://localhost:4174 にアクセスし、admin / admin でログイン。
  5. 初期データ投入 — Flow/Planner/NMOSのテストデータを登録して操作を慣らします。

Windows で PowerShell を使う場合、パスの区切り文字は自動的に処理されます。

トラブルシューティング

  • ポート競合 → .env の APP_PORT, UI_PORT を変更してください。
  • 権限エラー → Linux では sudo chown -R $USER:$USER certs pgdata を実行してください。
  • HTTPS証明書 → certs/server.crt と server.key を社内CAの証明書に差し替えることができます。
  • コンテナが起動しない → docker compose logs でエラーログを確認してください。
