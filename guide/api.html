REST API / MQTT

全機能はRESTで操作でき、JWT認証を利用します。MQTTを有効化すると mmam/flows/events などのトピックで更新を購読できます。

主要エンドポイント

  • POST /api/login
  • GET /api/flows/export / POST /api/flows/import
  • GET /api/logs?kind=api|audit

連携ヒント

  • REST PATCH + alias/user fieldsでメモ共有
  • MQTTを監視してBPM/監視システムへ転送
  • Automation結果をIssue Trackerに登録
